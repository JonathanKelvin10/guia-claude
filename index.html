Claude — O Chat com IA

A forma mais simples e acessível de usar a inteligência artificial da Anthropic. Basta digitar e conversar.

💬 Claude Chat
💬

O que é o Claude?

O Claude é um assistente de IA que você acessa pelo navegador (claude.ai), pelo celular ou pelo computador. Você conversa em texto — como num chat — e o Claude responde, cria, explica, resume ou ajuda com qualquer tarefa intelectual.

  • ✓ Responde perguntas em qualquer assunto
  • ✓ Escreve textos, e-mails, relatórios, artigos
  • ✓ Lê e resume documentos (PDF, Word, etc.)
  • ✓ Analisa imagens e planilhas
  • ✓ Ajuda com código e matemática
  • ✓ Traduz, revisa e melhora textos
🎯 Para quem é?

Qualquer pessoa! Estudantes, profissionais, curiosos. Não precisa de conhecimento técnico. Se você sabe digitar, você usa o Claude.

🚀 Como começar em 3 passos

1

Acesse claude.ai

Entre no site pelo navegador ou baixe o app no celular. Crie uma conta gratuita com seu e-mail.

2

Digite sua pergunta ou pedido

Escreva como você falaria com uma pessoa. Não precisa de comandos especiais. Quanto mais detalhes, melhor a resposta.

3

Refine e continue a conversa

Se a resposta não foi exatamente o que queria, diga "ajusta para ficar mais curto" ou "agora mais formal". O Claude lembra do contexto.

Cowork — IA no seu Desktop

O Cowork é o Claude com superpoderes instalado no seu computador. Ele cria arquivos, navega na internet e executa tarefas complexas por você.

🖥️ Cowork Desktop
🖥️

O que é o Cowork?

É uma versão do Claude que roda como aplicativo no seu computador (Windows/Mac). A diferença? Ele não só responde — ele age. Cria arquivos reais, navega em sites, organiza pastas, preenche formulários e muito mais.

  • ✓ Cria documentos Word, Excel, PowerPoint e PDF
  • ✓ Navega na internet e pesquisa por você
  • ✓ Acessa e organiza arquivos do seu computador
  • ✓ Usa Skills (plugins) para expandir capacidades
  • ✓ Conecta com apps como Slack, Asana, Gmail
  • ✓ Agenda tarefas automáticas recorrentes
⚡ Diferença chave

Claude Chat vs Cowork

💬 Claude Chat diz:

"Aqui está o conteúdo do relatório que você pode copiar e colar no Word..."

↓

🖥️ Cowork faz:

"✅ Relatório criado! Clique aqui para abrir o arquivo .docx"

🎯 Para quem é?

Profissionais, empreendedores, pessoas que trabalham muito com documentos, planilhas e apresentações. Não precisa saber programar.

O que o Cowork consegue fazer

📊

Criar planilha completa

"Cria uma planilha de controle financeiro pessoal com gráficos, fórmulas e formatação profissional"

📑

Gerar relatório em Word

"Pesquisa sobre o mercado de energia solar no Brasil e cria um relatório de 5 páginas em .docx"

🎨

Montar apresentação

"Cria uma apresentação de 10 slides sobre nosso novo produto para a reunião de vendas amanhã"

🔍

Pesquisar na internet

"Pesquisa os 5 melhores softwares de gestão de projetos em 2025 e compara os preços"

📂

Organizar arquivos

"Olha a pasta Downloads e organiza os arquivos por tipo em subpastas separadas"

⏰

Tarefa automática diária

"Todo dia às 8h, resume as notícias de tecnologia e me manda um briefing por e-mail"

💡 Dica de ouro

O Cowork tem "Skills" (habilidades extra) que você pode instalar — como plugins que ampliam o que ele pode fazer. Por exemplo, uma Skill para criar PDFs profissionais, outra para criar slides no PowerPoint, e assim por diante.

Claude Code — IA no Terminal

Para programadores e desenvolvedores que querem delegar tarefas de código para a IA diretamente no terminal, sem sair do fluxo de trabalho.

⌨️ Claude Code
⌨️

O que é o Claude Code?

O Claude Code é uma ferramenta de linha de comando (CLI) que traz a IA do Claude para dentro do terminal do desenvolvedor. Em vez de copiar código do chat, o Claude Code lê, escreve e executa código no seu projeto diretamente.

  • ✓ Lê e entende todo o seu projeto de código
  • ✓ Escreve funções e componentes completos
  • ✓ Corrige bugs automaticamente
  • ✓ Roda testes e valida o código
  • ✓ Faz commits no Git por você
  • ✓ Refatora código legado
🎯 Para quem é?

Desenvolvedores e programadores que usam terminal. Requer conhecimento técnico básico em programação e linha de comando.

Desenvolvedores relatam economizar 2 a 4 horas por dia usando Claude Code para tarefas repetitivas de código.

O que o Claude Code faz na prática

🐛

Caça bugs automaticamente

"Meu app está travando na tela de pagamento" → Claude lê o código, acha o erro e corrige sozinho

🏗️

Constrói features inteiras

"Cria um sistema de autenticação com JWT, login, registro e recuperação de senha"

🧪

Escreve testes automatizados

"Gera testes unitários e de integração para todas as funções da pasta /api"

🔄

Refatora código antigo

"Esse código tem 3 anos, refatora para usar os padrões modernos do React 2025"

📖

Documenta automaticamente

"Adiciona comentários e documentação JSDoc em todas as funções do projeto"

🚀

Configura deploy

"Cria os arquivos Docker e o pipeline de CI/CD no GitHub Actions para esse projeto"

⌨️ Como instalar o Claude Code

1

Tenha Node.js instalado

O Claude Code requer Node.js 18+ no seu sistema. Verifique com node --version no terminal.

2

Instale via npm

No terminal, rode: npm install -g @anthropic-ai/claude-code

3

Entre na pasta do projeto e rode

Navegue até seu projeto e digite claude. Faça login com sua conta Anthropic e pronto!
